# Simple Clock — Documentation

## Overview

This is a very simple web page that displays the current system time and updates every second.

## Files

* `index.html` — Page structure
* `style.css` — Basic styling
* `script.js` — Time update logic

## How It Works

* The page loads and shows a heading and an empty time area.
* JavaScript gets the current time from your system.
* The time is updated on the page every second.

## How to Run

1. Create the three files listed above.
2. Put them in the same folder.
3. Open `index.html` in any web browser.

## Output

You will see the current time displayed on the screen, updating every second.
