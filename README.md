# Week 3 Assignment
Tasks:
* Create a public GitHub repository named devops-git-basics. (Do NOT initialize it with any files.)
* ⁠On your local machine:
* Create a folder for your project.
* Initialize it as a Git repository.
* Inside the folder:
* Create a README.md file and write a short personal bio (2–3 lines).
* Create a version.txt file with the content:

v1.0.0

* Stage and commit both files.
* Connect your local repository to the remote GitHub repository.
* Push your commit to GitHub (to the main branch).
* Now update version.txt to:

v1.0.1

* Commit the change and push it to GitHub.

Deliverables:
* GitHub repository link
